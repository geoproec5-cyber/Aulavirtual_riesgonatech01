# Ecosistema HTML - Curso Gestión de Riesgos Naturales

Versión GitHub Pages del ecosistema modular para el curso **Gestión de Riesgos Naturales Aplicada a Empresas Privadas**.

## Páginas incluidas

- `index.html` - Entrada general del curso.
- `solicitud-acceso.html` - Formulario F0 con consentimientos.
- `aula.html` - Aula del facilitador con burbuja, cámara local opcional y guía de tiempo.
- `pizarra.html` - Pizarra interactiva con dibujo, texto, figuras y descarga PNG.
- `laboratorio.html` - Laboratorio de escenarios de riesgo natural, empresarial y NaTech.
- `modulos.html` - Malla del curso y horas certificables.
- `herramientas.html` - Kobo, Sheets, Drive, Docs, ArcGIS, IA y ALOHA.
- `evaluacion.html` - Ecosistema de formularios Kobo + Google Sheets.
- `dashboard.html` - Panel de avance y competencias del estudiante.
- `consulta-gpt.html` - Panel de prompts guiados.
- `perfil-servicios.html` - Perfil profesional, paper NaTech y cartera de servicios.
- `admin.html` - Vista interna de cohorte y control académico.
- `certificados.html` - Generador demo de certificado Sambies listo para firma.
- `repositorio.html` - Recursos, plantillas, matrices y herramientas.

## Publicar en GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir todos los archivos y carpetas de este paquete a la raíz del repositorio.
3. Ir a **Settings > Pages**.
4. En **Build and deployment**, seleccionar **Deploy from a branch**.
5. Seleccionar la rama `main` y carpeta `/root`.
6. Guardar y esperar la URL pública de GitHub Pages.

## Notas técnicas

- El audio oficial de clases se maneja con Zoom.
- La cámara local del aula usa permisos del navegador y funciona mejor en HTTPS, por eso GitHub Pages es adecuado.
- Los formularios Kobo están representados como estructura funcional. En producción se deben reemplazar botones por enlaces reales de Kobo.
- La consolidación Google Sheets, Apps Script, Drive y Docs queda como siguiente fase de integración.
- La validación QR del certificado corresponde al sistema de Sambies; aquí se genera el insumo/plantilla listo para revisión y firma.
